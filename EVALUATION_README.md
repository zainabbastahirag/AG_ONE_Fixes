| File | Use |
|------|-----|

| `evaluation_dashboard.html` | Browser dashboard — scores auto-save on your device |
| `generate_evaluation_excel.py` | Regenerate the Excel if you change names/criteria |

## How to use the Excel file

1. Open `candidate_evaluation.xlsx`
2. **Phase 1 Scores** / **Phase 2 Scores** — enter numbers 1–5; Total and Average calculate automatically
3. **Phase 1 Notes** / **Phase 2 Notes** — write detailed comments per person
4. **Summary** — quick comparison, strengths, recommend yes/maybe/no

## How to use the HTML dashboard

1. Open `evaluation_dashboard.html` in Chrome / Edge / Firefox (double-click the file)
2. Use tabs: **Session 1**, **Session 2**, **Summary**
3. Click **Save locally** to store on your computer
4. Click **Screenshot view** to hide buttons for a clean screenshot
5. **Export JSON** to back up your data
